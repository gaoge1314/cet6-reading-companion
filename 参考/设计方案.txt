

## 修正后的做题顺序

```
看选项（5道题全部题干+选项）
    ↓  分析：讨论什么主题？情感正向/负向？作者大致态度？
    ↓
读首段 + 各段首句 + 尾段
    ↓  基于已读信息修正猜测，确定/接近中心
    ↓
中心做题（能做先做，排除与中心矛盾的）
    ↓
定点精读（剩余题目定位回原文）
    ↓
核对答案 + 错题分析
    ↓
全篇复盘
```

核心原则：**分析必须基于当前阶段已读到的内容**，不跳步、不剧透。

## 第二部分：各 Phase 主区域内容设计

### Phase 1 —「选项预判」
> 此时还没看文章，只看题干和选项

| 内容 | 说明 |
|:--|:--|
| 题干关键词提取 | 5道题分别问什么（pursue success / radical individualism / assumption / success stories / infer） |
| 主题猜测 | 从选项反推：反复出现 friends / success / individual → 友谊与成功的关系 |
| 情感色彩预判 | 选项中的措辞："ignore ties" "detrimental effects" "neglect close friends" → 整体**偏负面/批判性** |
| 作者态度预判 | 选项中"reexamine harm" "cannot afford to neglect" → 作者**倾向于重视友谊** |

<small>（以上全部从选项推断，还没看到文章一句原文）</small>

### Phase 2a —「首尾定中心」
> 已读：首段 + 各段首句 + 尾段

| 内容 | 说明 |
|:--|:--|
| 首尾句列表 | 可划词（保持不变） |
| 情感色彩修正 | 对比 Phase 1 的预判：P1尾"lose friends" → 确认负面；P8"We cannot be complacent" → 作者态度清晰了 |
| 中心思想框 | 基于首尾句推断：**追求成功不应牺牲友谊，朋友是锚不是代价** |
| 出题位置预测 | 首段、各段首尾句大概率直接出题（老师技巧） |

### Phase 2b —「中心做题」
> 已有中心，逐题用中心排除 + 标记待精读题

| 内容 | 说明 |
|:--|:--|
| 逐题判断 | 哪些题中心能做？哪些需要精读？ |
| 中心做题理由 | 例如46题：选项A/B/C都与"友谊=温暖"相关，D说"忽视友谊"才符合"lose friends"中心 |
| 待精读标记 | 哪些题需要回原文定位（标注★） |

### Phase 2c —「定点精读」
> 精读定位段，解决剩余题目

| 内容 | 说明 |
|:--|:--|
| 定位段精读 | 显示对应段落，标注关键句 |
| 同义替换分析 | reevaluate→reexamine, harm→detrimental effects 等 |
| 情感再确认 | 精读后对情感色彩/中心有什么修正？ |

### Phase 3 —「核对答案」
> 揭晓答案 + 只分析实际选错的

| 内容 | 说明 |
|:--|:--|
| 分数 + 逐题结果 | ✅/❌ |
| 正确题理由 | 用原文句子佐证 |
| 错题分析 | 只显示用户实际选错的选项分析 |
| 翻译面板 | 左右对照（生词红色 + 解题句绿色） |

### Phase 4 —「全篇复盘」

| 内容 | 说明 |
|:--|:--|
| 技巧使用统计 | T1-T7用了多少次 |
| 各题技巧复盘 | 每道题用的什么方法 |
| 易错点提醒 | 关键区分点 |
| 导出按钮 | 生词本导出 |
