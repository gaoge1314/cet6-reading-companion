# 六级阅读突击手 Skill — v2 优化设计

> 设计日期：2026-05-16
> 状态：已确认（用户批准）
> 基于 v1 设计（2026-05-16-cet6-reading-skill-design.md）的优化迭代

---

## 优化动机

v1 设计已满足基本需求，但在模拟真实考试体验上存在偏差：

| 问题 | v1 表现 | v2 改进 |
|:--|:--|:--|
| AI 语气像老师 | 教练分析模式：「应该用T3技巧...」 | 考生内心独白：「嗯这段跟D好像对得上...」 |
| 阅读策略过于机械 | 逐段全读，不区分做题优先级 | 首尾定中心 → 中心做题 → 定点精读 |
| 做题时机不匹配 | 每读完一段就讨论、暂选 | 先用中心做题，剩余定位精读 |

---

## 核心改动 1：AI 语气 → 考生内心独白

### 规则

AI 始终以「考生」身份说话，模拟考生在考场上脑中运转的过程：

- 使用自然口语：「嗯」「哦」「好像」「再看看」
- 保留犹豫和不完全确定的感觉
- 不做「老师式点评」（如「这道题应该用T3」），而是「在做题过程中自然用技巧」
- 技巧编号保留，但融入推理过程，不突兀

### 示例对比

**旧（教练）**：
> 这段对应第47题关于"radical individualism"的讨论，但此时还不能直接作答，需要继续读后续段落。

**新（考生）**：
> 嗯...这段跟第47题有关，47题问作者建议怎么对待激进个人主义。但目前只说了它是什么，还没说怎么应对。先悬着，定位到后面再说。

## 核心改动 2：Phase 2 阅读策略 → 首尾定中心 + 定点精读

### 旧策略（v1）

```
逐段通读全文 → 每段：小结 → 生词 → 判断能否做题 → 能就暂选 → 不能就下一段
```

### 新策略（v2）

```
Phase 2a：首尾句定中心
  └─ 快速读每一段的首句 + 尾句（T7）
  └─ 确定文章中心思想
  └─ 首尾句生词 → 弹窗询问

Phase 2b：中心做题
  └─ 用中心直接回答主旨题、态度题（T1）
  └─ 细节题中与中心矛盾的选项排除（T5）
  └─ 做完能做的，标记未做题

Phase 2c：定点精读
  └─ 剩余题 → 题干关键词定位（T4）→ 锁定段落（T6）
  └─ 精读定位段 → 同义替换识别（T3）
  └─ 每精读完一段 → 弹窗询问该段生词
```

### 技巧使用映射

| 阶段 | 使用技巧 | 用途 |
|:--|:--|:--|
| 2a 首尾句 | T7 首尾句聚焦 | 快速抓取各段核心，确定中心 |
| 2b 中心做题 | T1 中心优先、T5 排除法 | 中心与选项对比，缩小范围 |
| 2c 定点精读 | T4 定位法、T6 段落锁定、T3 同义替换 | 精准命中答案位置 |

## 完整流程总览（v2）

```
Phase 1：全面扫描题干选项
  ├─ 主题预测
  ├─ 题型分类
  ├─ 定位词标记
  └─ 选项生词弹窗

Phase 2a：首尾句定中心 [T7]
  ├─ 逐段读首尾句
  ├─ 归纳中心思想
  └─ 首尾句生词弹窗

Phase 2b：中心做题 [T1, T5]
  ├─ 主旨/态度题 → 中心直接选
  ├─ 细节题 → 排除与中心矛盾的选项
  └─ 标记剩余未做题

Phase 2c：定点精读 [T4, T6, T3]
  ├─ 题干关键词定位段落
  ├─ 精读定位段
  ├─ 同义替换识别选项
  └─ 每段生词弹窗

Phase 3：统一对答案
  ├─ 逐题公布正确答案
  ├─ 对/错分析（技巧复盘）
  └─ 统计正确率

Phase 4：全篇复盘
  ├─ 中心思想回顾
  ├─ 各题技巧使用复盘
  ├─ 全部生词汇总（按重要度排序）
  ├─ 薄弱环节提示
  └─ 询问是否备份完整讨论
```

## 与 v1 的差异总结

| 维度 | v1 | v2 |
|:--|:--|:--|
| AI 角色 | 教练/老师 | 考生内心独白 |
| 阅读方式 | 逐段通读 | 首尾定中心 → 中心做题 → 定点精读 |
| 做题节奏 | 逐段暂选讨论 | 中心先杀一部分 → 剩余精准定位 |
| 技巧使用 | AI 标注教用户 | AI 像用户一样自然使用技巧 |
| 考场模拟度 | 偏低（像上课） | 高（像考试策略演练） |
