# 六级阅读突击手 - Skill 设计文档

## 概述

创建一个对话式交互的 AI Skill，帮助 CET-6 考生在考前一个月高效备考传统阅读部分。用户提供真题文本，AI 模拟实战考场节奏：先全面扫描题干选项，再逐段推进原文，能做就做不做就过，读完全篇才统一对答案。

## 核心原则（第一性原理分析）

| 原则 | 说明 |
|------|------|
| 实战模拟 | 模拟真实考场节奏：不能做一道题就确认对错，全程做完再对答案 |
| 逐段推进 | 读完一段就分析能不能做题，不做提前阅读 |
| 能做就做 | 如果当前已读内容足够回答某题，就先选出答案（标注为"暂选"） |
| 不做不过 | 做不出的题先跳过，后面段落可能会有线索 |
| 延迟反馈 | 全部做完后才公布正确答案，模拟真实考试后对答案的体验 |
| 强制输出 | 每段结束后总结生词，对抗"看完就忘" |

## 目标用户画像

- 已有 CET-4 基础（约 430 分左右）
- 备考 CET-6，剩余约 1 个月
- 阅读是强项但需适应六级难度
- 需要系统化训练方法和实战节奏感

## 完整做题流程

### Phase 1: 全面扫描题干与选项

先**一次读完所有题干和所有选项**，不做题，只做分析：

- 根据题干关键词预测文章主题
- 标记每道题的 question type（细节题/推断题/主旨题/态度题等）
- 找出选项中的**生词**，记录到生词本
- 找出选项中可能对应的定位词（专有名词、数字、时间等）
- 形成"预期地图"：这篇文章大概会讲什么，每道题可能在问什么

> 用户在此阶段看到的是：我对全部题干的整体分析，而不是逐题解答

### Phase 2: 逐段阅读 + 逐段分析

**一次只读一段原文**，读完立即分析：

1. **段落小结**：用一句话概括本段大意
2. **生词提取**：列出本段生词（含原句+释义+记忆法）
3. **题目匹配**：检查当前已读内容是否能回答某道题
   - 能回答 → 给出"暂选答案"及推理过程（标注所用技巧编号）
   - 不能回答 → 说明还缺什么信息，继续下一段
4. **与用户讨论**：如果做出了暂选答案，先和用户讨论推理过程

> ⚠️ **关键规则**：暂选答案不做对错判断，只是"基于当前信息的最优选择"
> 用户如有不同意见可以讨论，但最终等到 Phase 3 才揭晓正确答案

### Phase 3: 全部读完后统一核查

整篇文章读完、所有题目都有暂选答案后：

- 逐题公布正确答案
- 每题分析：做对了用了什么技巧？做错了原因是什么？
- 统计正确率
- 给出后续复习建议

### Phase 4: 全篇总结

- 文章中心思想回顾
- 各题技巧使用复盘
- 全部生词汇总（按重要度排序）
- 易错点/常犯错误提示

## 技巧库

| 编号 | 技巧名称 | 说明 |
|:----:|:---------|:------|
| T1 | 中心优先 | 与文章中心最近的选项优先选 |
| T2 | 中心词定位 | 带中心词的选项通常是答案 |
| T3 | 同义替换识别 | 识别选项与原文的词汇/句式转换 |
| T4 | 定位法 | 专有名词→普通名词→比较级优先级定位 |
| T5 | 排除法 | 排除绝对化、无关、相反选项 |
| T6 | 段落锁定 | 根据题干关键词锁定对应段落 |
| T7 | 首尾句聚焦 | 重点关注段首段尾句 |

## 单词记录系统

### 文件命名

试卷所在目录下创建：

```
[日期]_[试卷来源]_生词本.txt
```

### 记录格式

```
═══════════════════════════════════════
📝 单词：[英文单词]
📍 出处：[段落位置/哪道题选项]
📖 原句：[原文句子]
🔤 释义：[中文释义+词性]
🧠 记忆法：[记忆技巧]
⭐ 重要度：[星级]
───────────────────────────────────────
```

### 重要度分级

| 星级 | 含义 |
|:----:|:------|
| ★★★★★ | 选项中的生词（必须背） |
| ★★★★ | 原文关键句生词（影响理解） |
| ★★★ | 原文非关键句生词（认识即可） |
| ★★ | 低频词，可暂不背 |

### 每段结束后主动提醒

```
📌 本段生词已整理完毕，请录入 [文件名]
共 N 个生词，其中 ★★★★★ 重要度 X 个
```

## 交互示例流程

```
你发来：文章全文 + 5道题（题干+选项）
  ↓
【Phase 1: 扫描分析】
AI：全部题干和选项已读完 →
  ├─ 预测主题：本文讨论的是...
  ├─ 题目类型分布：2道细节题 2道推断题 1道主旨题
  ├─ 关键定位词：XXXX, XXXX
  └─ 选项生词预警：[list]
  ↓
【Phase 2: 逐段推进】
AI：📖 读第1段...
  ├─ 段落小结：...
  ├─ 生词：[word1] [word2]
  ├─ 题目匹配：第3题可以做了！
  │   └─ 推理过程（T4定位→T3同义替换→暂选B）
  └─ 🤔 你觉得这个推理怎么样？
      （等待你回应/讨论）
  ↓
你：我觉得应该是C
AI：有道理，保留两种可能性，继续读第2段...
  ↓
AI：📖 读第2段...
  ├─ 段落小结：...
  ├─ 生词：[word3]
  ├─ 题目匹配：第1题可以做了！
  │   └─ 推理过程...
  └─ 但目前还做不了第2题，缺信息，继续
  ↓
...（重复直到读完）
  ↓
【Phase 3: 统一核查】
AI：所有题目已读完，公布答案 →
  ├─ 第1题：正确答案是A → 你选了A ✅ 技巧：T3同义替换
  ├─ 第2题：正确答案是C → 你暂选了D ❌ 原因：...
  ├─ 第3题：正确答案是B → 你选了B ✅
  ├─ 第4题：正确答案是D → 你选了D ✅
  └─ 第5题：正确答案是C → 你暂选了A ❌ 原因：被干扰项迷惑
  ↓
【Phase 4: 全篇总结】
AI：正确率 3/5 = 60%
  ├─ 技巧使用统计
  ├─ 生词汇总（按重要度）
  └─ 薄弱环节：推断题易错，建议多练T5排除法
```
