# 🎯 六级阅读突击手

> CET-6 传统阅读（Section C）实战陪练 — 模拟考场内心独白，用老师的技巧做题
> **v3 新特性：HTML 阅读器 + 划词记录生词**

## 这是什么？

这是一个 **Trae AI Skill**，专门用于六级传统阅读训练。它不像普通 AI 那样直接给你答案，而是：

- 🧠 **模拟你的大脑** — 用考生内心独白的方式展示做题全过程
- 📚 **使用老师教的 7 大技巧** — T1~T7 自然融入推理
- 🌐 **HTML 阅读器** — 在浏览器中舒适阅读文章，选中单词即可加入生词本
- 📌 **自主划词记生词** — 你自己选择哪些单词不认识，不需要 AI 去猜
- 💾 **每篇单独备份** — 所有讨论和推理过程保存为 Markdown 文件

## 快速开始

### 1. 安装 Skill

在 Trae 中搜索并安装 `cet6-reading-companion` 技能，或手动将 SKILL.md 放入对应目录。

### 2. 准备试题

将六级真题 PDF 放入 `题目/` 目录。备份和生词本会**自动跟随 PDF 的位置**生成：

```
题目/25年12月真题+解析+听力/2025年12月六级/
├── 2025.12六级真题第1套.pdf              ← 你的原始试题
└── 2025.12六级真题第1套/                  ← 自动创建
    └── 传统阅读/
        ├── 备份_[日期].md
        └── [日期]_生词本.md
```

无需手动创建文件夹，AI 会根据 PDF 路径自动处理。

### 3. 开始练习

在聊天框里：
1. **发送** PDF 的 Section C 部分（文章 + 5 道题的原文）
2. AI 立即进入 **Phase 1** — 扫描所有题干和选项
3. AI **生成 HTML 阅读器文件**（`cet6-reader.html`），双击在浏览器中打开
4. 跟随指引，在浏览器中阅读，**划词记录生词**，一步步完成整篇练习

### 4. HTML 阅读器用法

```
📖 cet6-reader.html 功能说明：

┌─ 顶部导航 ─────────────────────────────┐
│ Phase 1 | 2a | 2b | 2c | 3 | 4       │  ← 点击切换阶段
├──────────────────────┬─────────────────┤
│                      │  📝 生词本      │
│   主内容区            │  (划词即加)     │
│   (当前 Phase 内容)   ├─────────────────┤
│                      │  📋 答题卡       │
│                      │  (Phase 3显示)   │
└──────────────────────┴─────────────────┘

🖱️ 选中任意单词/短语 → 弹出"加入生词本"按钮
📥 右上角"导出 .md" → 一键导出 Markdown 生词本
```

### 5. 练习流程

```
Phase 1：全面扫描题干选项
  → 主题预测 → 题型分类 → 定位词标记
  → HTML Phase 1 页面查看完整题干
  → 划词记录生词

Phase 2a：首尾句定中心 [T7]
  → 扫每段首尾句 → 确定文章中心
  → HTML Phase 2a 页面查看首尾句

Phase 2b：中心做题 [T1, T5]
  → 用中心做主旨/态度题
  → 排除与中心矛盾的选项

Phase 2c：定点精读 [T4, T6, T3]
  → 题干定位 → HTML Phase 2c 精读相关段落
  → 同义替换识别

Phase 3：统一对答案
  → 逐题揭晓 → HTML Phase 3 查看正确答案

Phase 4：全篇复盘
  → 中心回顾 → 技巧复盘 → 生词汇总 → 导出 .md
```

## 7 大阅读技巧

| 编号 | 技巧 | 考场上怎么用 |
|:--:|:--|:--|
| **T1** | 中心优先 | 和文章中心最近的选项优先选 |
| **T2** | 中心词定位 | 带中心词的选项通常是答案 |
| **T3** | 同义替换识别 | 选项和原文同一个意思换了个说法 |
| **T4** | 定位法 | 专有名词→普通名词→比较级，逐级定位 |
| **T5** | 排除法 | 排除绝对化、无关、与中心相反的选项 |
| **T6** | 段落锁定 | 题干问哪段，就读哪段 |
| **T7** | 首尾句聚焦 | 段首句和段尾句是各段核心 |

## 生词本

### 两种记录方式

| 方式 | 说明 |
|:----|:------|
| **🖱️ HTML 划词** | 在浏览器中选中单词 → 弹出"加入生词本" → 自动存储到浏览器 |
| **📥 导出 .md** | 点击"导出 .md"按钮，一键生成 Markdown 生词本文件 |

### AI 补充释义

导出生词本后，告诉 AI 你导出了生词，AI 会在聊天框中帮你补充：

```
| 单词 | 释义 | 重要度 |
|:--|:--|:--:|
| detrimental | adj. 有害的 | ★★★★★ |
| endeavor | n. 努力 | ★★★★★ |
| confide | v. 倾诉 | ★★★★ |
```

## 设计理念

- 🎯 **实战模拟** — AI 模拟考生心里怎么想，不是老师怎么教
- ⏱ **延迟反馈** — 读完全篇才公布答案，训练完整做题能力
- 🌐 **HTML 阅读器** — 在浏览器中舒适阅读，长文章不再拥挤
- 🖱️ **自主划词** — 你自己选择生词，不需要 AI 去猜
- 🔄 **技巧内化** — 7 大技巧在推理中自然使用，形成肌肉记忆
